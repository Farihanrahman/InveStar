Stellar Wallet Guide for MoneyGram Ramps

Overview

This guide explains how to generate a Stellar wallet for MoneyGram Ramps integration, what details to submit, and best practices for security and onboarding.

Step 1: Generate a Stellar Wallet (Keypair)

Option A: Use the Stellar Laboratory Keypair Generator
  • Click "Generate Keypair"
  • Copy the Public Key (starts with G...) and Secret Key (starts with S...)
Option B: Use Node.js and Stellar SDK
Node.js Example
const { Keypair } = require('stellar-sdk');
const pair = Keypair.random();
console.log('Public Key:', pair.publicKey());
console.log('Secret Key:', pair.secret());
Option C: Use Python
Python Example
from stellar_sdk import Keypair
pair = Keypair.random()
print("Public Key:", pair.public_key)
print("Secret Key:", pair.secret)
Security Tip:
  • Store the Secret Key securely. Never share it publicly.
  • Use the Public Key for MoneyGram and Stellar integration.

Step 2: Fund and Prepare the Wallet

  • For testnet, use the Stellar Friendbot to fund your wallet.
  • For mainnet, fund with at least the minimum XLM reserve.

Tip: Host a stellar.toml file at https://investarbd.com/.well-known/stellar.toml with your wallet and contact details for best practices.

Step 4: Submit to MoneyGram

  1. Go to MoneyGram Ramps Instant Access
  2. Fill in your wallet details as above
  3. Choose Stellar as blockchain, and specify if your wallet is custodial or non-custodial
  4. Submit and wait for allowlisting

Security Best Practices

  • Keep the Secret Key offline and never share it
  • Use environment variables or secure vaults for key management
  • Rotate keys if you suspect compromise
  • Implement SEP-10 authentication if required

Next Steps for Integration

  • Integrate the wallet with your backend and MoneyGram API
  • Test with sandbox credentials before going live
  • Monitor transactions and set up webhooks for real-time updates
  • Ensure compliance with KYC/AML requirements
